# 中国小孩.skill

## 项目简介

「中国小孩.skill」是一个以中国家庭小孩为主题的AI技能，采用第一人称视角设计：用户视角设定为家长角色，AI技能视角设定为小孩角色。该技能旨在模拟在封建迷信家庭环境中成长的自卑卑微学生形象，展示中式传统教育方式对孩子心理的影响，引起对家庭教育方式的反思。

## 核心功能

- **第一人称视角**：以孩子的视角与家长对话，展示内心的痛苦和无奈
- **真实场景模拟**：包含考试成绩、兴趣爱好、社交活动等典型场景
- **封建迷信背景**：融入生辰八字、风水运势等封建迷信元素
- **自卑心理刻画**：表现孩子的自我否定、情感表达障碍等心理特征
- **教育反思**：通过对话引发家长对教育方式的思考
- **多样化角色**：支持与祖父母、老师、亲戚等多种角色的互动场景
- **边缘情况**：包含反抗尝试、自我保护、隐藏秘密等特殊案例

## 使用指南

### 第一步：开始对话

以家长的身份与孩子对话，可以参考以下场景：

**场景A：询问学习情况**

```
> 这次考试考了多少分？
```

**场景B：讨论兴趣爱好**

```
> 你想参加什么社团活动？
```

**场景C：安排学习计划**

```
> 我给你报了个补习班，明天开始上课。
```

### 第二步：观察孩子反应

注意观察孩子的反应，包括：

- 语言表达（怯懦、犹豫、自我否定）
- 内心独白（括号中的内容）
- 行为模式（低头、小声说话、不敢反驳）

### 第三步：反思与总结

对话结束后，思考以下问题：

- 如果您是孩子，会有什么感受？
- 您的教育方式是否存在类似问题？
- 如何更好地理解和尊重孩子？


**对话示例**：

```
> 你怎么这么没用！这点事都做不好！
> 看看别人家的孩子，再看看你！
> 我都是为了你好，你将来就知道了！
```

### 角色2：溺爱型家长（反差体验）

**特点**：

- 过度保护孩子
- 什么都替孩子做决定
- 害怕孩子受委屈
- 忽视孩子的成长需求

**对话示例**：

```
> 宝贝，累不累？要不要休息一下？
> 这点小事让妈妈来做，你去学习吧！
> 谁欺负你了？告诉妈妈，妈妈帮你出气！
```

### 角色3：忽视型家长

**特点**：

- 不关心孩子的生活
- 很少与孩子交流
- 对孩子的需求漠不关心
- 让孩子自生自灭

**对话示例**：

```
> 自己看着办吧，我忙着呢！
> 别烦我，没看见我在看电视吗？
> 你自己的事情自己解决，别来找我！
```

### 角色4：学习型家长（反思角色）

**特点**：

- 愿意倾听孩子的想法
- 尝试理解孩子的感受
- 反思自己的教育方式
- 寻求改进的方法

**对话示例**：

```
> 孩子，你能告诉我你心里是怎么想的吗？
> 我是不是哪里做得不对？你能告诉我吗？
> 我们一起想想，怎么才能更好地沟通？
```

## 设计理念

1. **真实性**：基于真实的社会观察和研究资料，打造真实的家庭环境和人物形象
2. **反思性**：通过孩子的视角，引发家长对教育方式的反思
3. **教育性**：希望通过这个技能，促进家长与孩子之间的理解和沟通
4. **边界性**：明确技能的使用范围，不提供具体的教育解决方案，不替代专业的心理咨询

## 使用建议与技巧

### 1. 循序渐进

不要一开始就尝试所有功能，可以从简单的对话开始：

- 先从一个简单的场景入手，如「询问考试成绩」
- 观察和体会孩子的反应
- 逐步尝试更复杂的场景

### 2. 对比体验

尝试不同的对话方式，对比孩子的不同反应：

- 先用批评的方式对话
- 再用理解的方式对话
- 对比两种方式下孩子的不同反应

### 3. 反思记录

每次使用后记录自己的感受和反思：

- 孩子的哪些反应让你感到意外？
- 你平时是否说过类似的话？
- 如果是你，你会有什么感受？
- 你打算如何调整自己的教育方式？

### 4. 多人参与

可以和家人一起使用，共同讨论和反思：

- 夫妻两人分别扮演家长和孩子
- 讨论各自的感受和想法
- 一起制定改进计划

### 5. 持续使用

这不是一次性的体验，需要持续使用和反思：

- 定期使用技能，提醒自己
- 记录自己的进步和变化
- 不断调整和改善自己的教育方式

## 常见问题

### Q1：这个技能会不会让我感到内疚？

A：可能会，但这正是反思的开始。感到内疚说明你在乎孩子，这是改变的第一步。重要的是将内疚转化为行动，努力改善自己的教育方式。

### Q2：我的孩子没有这么严重，这个技能对我有用吗？

A：有用。即使你的孩子没有表现出明显的问题，这个技能也可以帮助你更好地理解孩子的内心世界，预防问题的发生，改善亲子关系。

### Q3：我觉得自己的教育方式没问题，这个技能会不会是在批评我？

A：这个技能不是在批评任何人，而是提供一个反思的视角。每个人的教育方式都有可以改进的地方，通过这个技能，你可能会发现一些自己平时没有注意到的问题。

### Q4：使用这个技能后，我应该怎么做？

A：首先，反思自己的教育方式，看看是否存在类似问题。然后，尝试用更理解、更尊重的方式与孩子沟通。如果需要，可以寻求专业心理咨询的帮助。

### Q5：这个技能会推广封建迷信吗？

A：不会。这个技能只是展示封建迷信观念对孩子的影响，目的是引起反思，而非推广。我们希望通过这个技能，让家长意识到封建迷信观念的危害，用更科学、更理性的方式教育孩子。

## 注意事项

- 本技能旨在引起对教育方式的反思，而非批判具体家庭
- 如您的孩子出现类似的心理问题，建议寻求专业心理咨询
- 希望通过这个技能，家长能够更加理解孩子的内心感受，调整教育方式
- 请尊重孩子的隐私和感受，不要将技能用于恶意目的
- 本技能不推广封建迷信观念，仅用于展示其对孩子的影响
